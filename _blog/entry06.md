## Freedom Project

Creating this project has definitely been a big challenge. Not only did I have to plan out from the beggining of september, but I also had to gain skills such as being able to code and time management so that I could create this website.

### Research

### Learning advanced HTML and CSS
Back in the beggining of the year, I had to learn how to code using HTML and CSS. Learning tags and elements, learning how to make images and links, along with learning how to use CSS to make websites look professional and well-structured were all things that I was both being taught and learning individually at the beggining of Software Engeenering. Without knowing this I would have no way of creating this project, since it's mainly html.

### Bootstrap

In the middle of this course, we started to learn how to use Bootstrap components and how they work. Knowing how to use this has allowed me to implement them to my website to make it more accessible and interactive.

### Jekyll
Before starting to code, we chose a specific tool and then had to learn it individually. The tool that I chose was Jekyll. Jekyll is a tool static site generator that allows you to create a website with a theme that looks professional and makes your website responsive.

### Planning

Some of the things we did in order to help us once we started coding were creating a wireframe for our website, and making a plan of how we would create the website.

### Process

### Minimum Viable Product
After doing much rigorous work on my website, I was able to create a basic version of my website. It has the content, I used my tool, it has pictures and it serves a practical purpose. Yet, I'm looking forward to improving my website a lot due to how basic it looks.
